Key Guards & Constants
MAX_WRITES = 3
After 3 successful write_file calls, subsequent writes are rejected with an error message sent to the model.
MAX_ITERATIONS = 20
Hard loop limit. Prevents infinite tool-call loops. Model sees all 20 turns of history.
MAX_TOOL_OUTPUT = 500
Tool results > 500 chars are truncated in history to save context tokens. Full output still sent to frontend.
